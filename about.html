Sobre

Ricardo Almendro Ruiz

Sou natual de Santo André grande São Paulo, porém moro em Artur Nogueira que é uma cidade próxima a cidade na região de Campinas.

Trabalhei por um bom período em São Paulo, porém optei por vir para o interior buscando uma melhor qualidade e equilíbrio de vida.

Sou apaixonado por desenvolvimento de sistemas, principalmente voltado para Web. Gosto muito de aprender novas tecnologias mas gosto mesmo é das tecnologias baseadas em Javascript.

Já trabalho na área há algum tempo principalmente com desenvolvimento backend, mas nos últimos anos venho investindo muito no aprendizado das tecnologias frontend que é algo que tem chamado muito a minha atenção.

Algumas tecnologias que já tive a oportunidade de trabalhar:

  • Java
  • Javascript
  • HTML
  • CSS
  • Angular
  • React
  • VueJS
